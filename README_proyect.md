- **Odoo–Sage synchronizer (Mar 13–25):**
  - Implemented a full local sync pipeline in Python:
    - Sage exports -> `_master_sage/`
    - Odoo exports -> `_master_odoo/`
    - Sync outputs -> `_master/`
  - Built and documented flows for:
    - **Customers** (primary address from `AddressTypeNumber = 0`, parity for country/state)
    - **Bill To contacts** (primary contacts + address, Odoo type = invoice)
    - **Delivery addresses** (non‑primary contacts + address, external ID `CustomerID_ContactRecordNumber`)
    - **Products** (products sync, NEW lists, and simplified Odoo import template)
